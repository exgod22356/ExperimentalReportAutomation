# ExperimentalReportAutomation 实验报告自动机  
## 功能  
实验报告心得自动生成。在命令行里输入关键词，自动生成一篇约一百到两百字的课程心得。  
## 使用方法  
直接执行/dist/ExperimentalReportAutomation.exe,或者命令行编译文件。输入关键词，复制输出即可。   
顺带一提，cmd中的复制是Enter键，粘贴为右键。将命令行里文字选中后按Enter即可复制。     
## 依赖环境  
python 3
